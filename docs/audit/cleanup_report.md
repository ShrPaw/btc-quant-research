## 10. Known Limitations

- Sample dataset is small (~1000 trades, 5 minutes)
- Larger raw datasets not committed to git
- No signal generation or trading logic
- No backtesting framework
- Cost model is reference-only
- Core pipeline uses stdlib only (no pandas/numpy despite Upwork skills listing them)
