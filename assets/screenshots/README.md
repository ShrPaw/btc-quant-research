# Portfolio Screenshots & Charts

Charts generated by `scripts/generate_portfolio_assets.py`.

## How to Generate

```bash
# Install matplotlib if needed
pip install matplotlib

# Generate sample data (if no real data available)
python scripts/run_pipeline.py

# Generate charts
python scripts/generate_portfolio_assets.py
```

## Charts

### Price Over Time (`price_over_time.png`)
Shows BTCUSDT price movement processed from raw trade data.
**Use in portfolio:** Demonstrates ability to process raw market data into clean time series.

### Volume Over Time (`volume_over_time.png`)
Shows buy vs sell volume aggregated from tick-level trades.
**Use in portfolio:** Shows understanding of market microstructure and trade classification.

### Net Delta (`delta_over_time.png`)
Shows per-second net delta (buy volume − sell volume).
**Use in portfolio:** Demonstrates ability to compute signed volume metrics.

### Cumulative Volume Delta (`cvd_over_time.png`)
Shows CVD — running sum of net delta tracking persistent buying/selling pressure.
**Use in portfolio:** Shows understanding of advanced market microstructure concepts.

### Rolling Volatility (`rolling_volatility.png`)
Shows realized volatility at 30s and 60s windows.
**Use in portfolio:** Demonstrates feature engineering and rolling statistics.

### Returns Distribution (`returns_distribution.png`)
Shows histogram of 1-second log returns.
**Use in portfolio:** Shows statistical analysis and distribution understanding.

### Feature Correlation (`feature_correlation.png`)
Shows correlation heatmap of key features.
**Use in portfolio:** Demonstrates analytical depth and feature validation.
